# 树状数组

## 树状数组简介

### 什么是树状数组

树状数组（Fenwick Tree）是一种巧妙的数据结构，它可以在O(logn)时间复杂度内完成单点更新和前缀和查询操作。想象你有一个需要频繁修改和统计的数据集合，比如实时股票价格，树状数组就像智能记账本，能快速记录变化并给出统计结果。

树状数组的核心在于理解其二进制索引结构，通过lowbit函数实现高效的更新和查询路径。关键知识点之间的关联如下：

1. 结构设计 → 二进制索引 → lowbit计算
2. 单点更新 → 向上传播 → 加法操作
3. 前缀查询 → 向下累积 → 减法操作
4. 应用扩展 → 离散化处理 → 多维变形

### 基本特点

- 高效性：更新和查询都只需O(logn)时间
- 空间经济：使用与原数组相同大小的存储空间
- 实现简单：核心代码仅需10行左右
- 适用范围：特别适合处理动态前缀和问题

### 与其他数据结构对比

| 数据结构 | 更新复杂度 | 查询复杂度 | 空间复杂度 | 适用场景 |
| -------- | ---------- | ---------- | ---------- | -------- |
| 普通数组 | O(1)       | O(n)       | O(n)       | 静态数据 |
| 前缀数组 | O(n)       | O(1)       | O(n)       | 只读数据 |
| 线段树   | O(logn)    | O(logn)    | O(4n)      | 复杂区间 |
| 树状数组 | O(logn)    | O(logn)    | O(n)       | 动态前缀 |

> 最适合场景：当需要频繁进行单点修改和前缀和查询时，树状数组是综合性能最优的选择

## 树状数组原理详解

### 结构揭秘（图示辅助理解）

想象将数组元素组织成多棵二叉树，但实际存储仍用数组实现。每个节点存储特定区间的和：

- 节点i管理区间：`[i - lowbit(i) + 1, i]`
- 例如节点8（二进制1000）管理8个元素
- 节点5（二进制0101）只管理自己

### lowbit函数解析

这个魔法函数是理解树状数组的关键：

```python
def lowbit(x):
    return x & -x
```

原理剖析：

- 二进制数取负是补码形式（取反加1）
- 例如：x=6(0110)，-x=1010，x&-x=0010=2
- 功能：找到x二进制表示中最低位的1

### 索引关系示例

以数组索引1-8为例：

| 索引 | 二进制 | 管理区间 | lowbit值 |
| ---- | ------ | -------- | -------- |
| 1    | 0001   | [1-1]    | 1        |
| 2    | 0010   | [1-2]    | 2        |
| 3    | 0011   | [3-3]    | 1        |
| 4    | 0100   | [1-4]    | 4        |
| 8    | 1000   | [1-8]    | 8        |

## 核心操作实现

### 初始化

```python
class FenwickTree:
    def __init__(self, size):
        self.n = size
        self.tree = [0] * (self.n + 1)  # 索引从1开始

    def update(self, idx, delta):
        while idx <= self.n:
            self.tree[idx] += delta
            idx += lowbit(idx)

    def query(self, idx):
        res = 0
        while idx > 0:
            res += self.tree[idx]
            idx -= lowbit(idx)
        return res
```

### 操作示例演示

假设初始数组为[0,1,2,3,4]（索引1-4）：

1. 单点更新：将索引2处值+3
    - 更新路径：2→4
    - tree +=3 → tree +=3
2. 前缀查询：求前3项和
    - 查询路径：3→2→0
    - sum = tree+tree

### 区间和计算

求区间[a,b]的和：

```python
def range_query(self, a, b):
    return self.query(b) - self.query(a - 1)
```

## 实战应用场景

### 经典案例：逆序对统计

给定数组[3,1,4,2]，统计逆序对数量：

```python
def count_inversions(arr):
    # 离散化处理
    sorted_arr = sorted(set(arr))
    rank = {v: i + 1 for i, v in enumerate(sorted_arr)}

    ft = FenwickTree(len(rank))
    inversions = 0
    for num in reversed(arr):
        inversions += ft.query(rank[num] - 1)
        ft.update(rank[num], 1)
    return inversions
```

执行过程：

1. 离散化得到rank字典：{1:1,2:2,3:3,4:4}
2. 逆序遍历原数组：
    - 处理2：查询rank1-1=1的前缀和（0），更新位置2
    - 处理4：查询rank3-1=3的前缀和（0），更新位置4
    - 处理1：查询0的前缀和（0），更新位置1
    - 处理3：查询rank2-1=2的前缀和（1），累计到结果

## 高阶应用与优化

### 多维扩展

二维树状数组实现：

```python
class FenwickTree2D:
    def __init__(self, rows, cols):
        self.row = rows
        self.col = cols
        self.tree = [[0] * (cols + 1) for _ in range(rows + 1)]

    def update(self, r, c, delta):
        i = r
        while i <= self.row:
            j = c
            while j <= self.col:
                self.tree[i][j] += delta
                j += lowbit(j)
            i += lowbit(i)

    def query(self, r, c):
        res = 0
        i = r
        while i > 0:
            j = c
            while j > 0:
                res += self.tree[i][j]
                j -= lowbit(j)
            i -= lowbit(i)
        return res
```

### 优化技巧

1. 批量初始化：O(n)时间建树

```python
def __init__(self, nums):
    self.n = len(nums)
    self.tree = [0] * (self.n + 1)
    for i in range(self.n):
        self.update(i + 1, nums[i])
```

1. 区间更新：结合差分数组思想

```python
def range_update(self, l, r, delta):
    self.update(l, delta)
    self.update(r + 1, -delta)
```

1. 最大值维护：修改更新策略

```python
def update_max(self, idx, value):
    while idx <= self.n:
        if value <= self.tree[idx]:
            break
        self.tree[idx] = value
        idx += lowbit(idx)
```
