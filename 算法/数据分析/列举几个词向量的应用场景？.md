# 列举几个词向量的应用场景？

问题：简单列举几个词向量的应用场景？

单词语义相关性

- 查找与给定词语意思最相近的 N 个词语：查找与当前词语的词向量的余弦相似度最大的 N 个单词
- 比较两个词语之间的相似度：计算两个词语的词向量之间的余弦相似度

词语类比

- 查找 “词语 A 对于词语 B 来说，就像词语 D 对于词语 C 一样” 关系中的词语 D：计算与词语 A 的词向量减词语 B 的词向量再加词语 C 的词向量的结果的余弦相似度最大的单词，例如
  king - man + women = queen。

短文本相似度

- 比较两端短文本之间的相似度：将短文本中的所有词向量求平均，从而将短文本表达为一个稠密向量（词袋模型的思想）
