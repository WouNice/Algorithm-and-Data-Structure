# 莫比乌斯反演（Möbius Inversion）

## 诞生背景与核心原理

### 莫比乌斯函数 μ(n) 的数学定义与组合意义

**定义**：莫比乌斯函数 μ(n) 是一个数论函数，其定义为：

```
      ⎧  1          n = 1
μ(n) = ⎨  0          n 有平方因子（即存在质数 p 满足 p² | n）
      ⎩  (-1)^k     n = p₁p₂...pₖ（k 个不同质因子的乘积）
```

**取值规律**：

| n | 质因数分解 | μ(n) |
|---|-----------|------|
| 1 | — | 1 |
| 2 | 2 | -1 |
| 3 | 3 | -1 |
| 4 | 2² | 0 |
| 5 | 5 | -1 |
| 6 | 2×3 | 1 |
| 7 | 7 | -1 |
| 8 | 2³ | 0 |
| 9 | 3² | 0 |
| 10 | 2×5 | 1 |
| 11 | 11 | -1 |
| 12 | 2²×3 | 0 |
| 13 | 13 | -1 |
| 14 | 2×7 | 1 |
| 15 | 3×5 | 1 |
| 30 | 2×3×5 | -1 |

**组合意义——容斥本质**：

莫比乌斯函数的核心意义在于它是**容斥原理的代数编码**。

考虑求 1 到 n 中与 m 互质的数的个数。根据容斥原理：
- 全体数：n
- 减去被每个质因子整除的：n/p₁, n/p₂, ...
- 加上被每两个质因子乘积整除的：n/(p₁p₂), ...
- 减去被每三个质因子乘积整除的：n/(p₁p₂p₃), ...
- …

每个项的系数正是 μ(d)——即 μ(d) 在容斥中扮演了"符号开关"的角色：
- 质因子个数为偶数时贡献正号（μ(d) = 1）
- 质因子个数为奇数时贡献负号（μ(d) = -1）
- 存在重复质因子时该项在容斥中不存在（μ(d) = 0）

因此：
```
         n
Σ μ(d) × ⌊ ─ ⌋ = 与 m 互质的数的个数
d|m       d
```

这正是莫比乌斯函数编码容斥原理的典范。

### 核心恒等式

```
        ⎧ 1,  n = 1
Σ μ(d) = ⎨
d|n     ⎩ 0,  n > 1
```

这个恒等式是整个莫比乌斯反演的根基。用符号表示：**∀n ≥ 1, Σ_{d|n} μ(d) = [n = 1]**，其中 `[P]` 是 Iverson 括号。

**证明**：
- n = 1：和式中只有 d = 1，μ(1) = 1 → 成立
- n > 1：令 n = p₁^{a₁}p₂^{a₂}...pₖ^{aₖ}，只需考虑无平方因子的 d（否则 μ(d) = 0）
  即 d 取 {p₁, p₂, ..., pₖ} 的所有子集的乘积
  设 d 恰含 i 个不同质因子，则 μ(d) = (-1)^i
  因此：
  ```
  Σ μ(d) = Σ C(k, i) × (-1)^i = (1 - 1)^k = 0
  d|n      i=0..k
  ```

### 莫比乌斯反演定理的两种形式推导

#### 除数形式（Divisor Form）

**定理**：若 F(n) = Σ_{d|n} f(d)，则 f(n) = Σ_{d|n} μ(d) × F(n/d)

**等价形式**：f(n) = Σ_{d|n} μ(n/d) × F(d)

**推导**：

```
设 F(n) = Σ f(d)
           d|n

右端 = Σ μ(d) × F(n/d) = Σ μ(d) × Σ f(k)
      d|n                   d|n     k|(n/d)

    = Σ Σ μ(d) × f(k)
      k|n  d|(n/k)

    = Σ f(k) × ( Σ μ(d) )
      k|n        d|(n/k)

    = f(n) × 1  +  Σ f(k) × 0   （核心恒等式：Σ_{d|m} μ(d) = [m=1]）
                     k|n, k≠n

    = f(n)  ✓
```

关键步骤说明：
1. 将 F(n/d) 按定义展开
2. 交换求和顺序：对 k|n 和 d|(n/k) 求和
3. 提取 f(k)，内层 Σ_{d|(n/k)} μ(d) 当 n/k = 1 即 k = n 时为 1，否则为 0

#### 倍数形式（Multiple Form）

**定理**：若 F(n) = Σ_{n|d} f(d)，则 f(n) = Σ_{n|d} μ(d/n) × F(d)

**推导**：

```
设 F(n) = Σ f(d)
           n|d

右端 = Σ μ(d/n) × F(d) = Σ μ(d/n) × Σ f(k)
      n|d                   n|d          d|k

令 t = d/n，则 d = nt，d|k ⇒ nt|k ⇒ t|(k/n)

    = Σ μ(t) × Σ f(k)
      t≥1        t|(k/n)

令 k = nk'，则：

    = Σ μ(t) × Σ f(nk') = Σ f(nk') × Σ μ(t)
      t≥1        t|k'       k'≥1         t|k'

    = Σ f(nk') × [k' = 1]   （核心恒等式）
      k'≥1

    = f(n)  ✓
```

#### 两种形式的关系

```
除数形式                   倍数形式
F(n) = Σ_{d|n} f(d)    F(n) = Σ_{n|d} f(d)
          ⇔                         ⇔
f(n) = Σ_{d|n} μ(d)F(n/d)  f(n) = Σ_{n|d} μ(d/n)F(d)
```

除数形式用于处理"因子和"关系，倍数形式用于处理"倍数和"关系。竞赛中最常用的是除数形式。

### 狄利克雷卷积视角——μ 作为 1 的逆元

**狄利克雷卷积定义**：

```
(f * g)(n) = Σ f(d) × g(n/d)
              d|n
```

卷积满足交换律、结合律，单位元为 ε(n) = [n = 1]。

**关键卷积关系**：

| 卷积 | 结果 | 说明 |
|------|------|------|
| μ * 1 = ε | μ(n) * 1(n) = [n = 1] | **μ 是常函数 1 的逆元** |
| φ * 1 = id | Σ φ(d) = n | 欧拉函数与恒等函数 |
| 1 * 1 = τ | Σ 1 = τ(n) | 约数个数 |
| id * 1 = σ | Σ d = σ(n) | 约数和 |
| 1 * μ = ε | 对称性 | 卷积交换律 |

**卷积视角下的莫比乌斯反演**：

```
F = f * 1    ⇔    f = F * μ
```

也就是说：
- 正向：F = f * 1（f 和 1 做卷积得到 F）
- 反向：f = F * μ（F 和 μ 做卷积还原 f）

由于 μ 是 1 的（卷积）逆元，两边同卷 μ 即可从 F 恢复 f。
```
F = f * 1
F * μ = (f * 1) * μ = f * (1 * μ) = f * ε = f
```

这正是"反演"的代数本质——**通过乘以逆元解除卷积**。

#### 常用数论函数表

| 符号 | 名称 | 定义 | 积性 |
|------|------|------|------|
| ε(n) | 单位函数 | [n = 1] | 是 |
| 1(n) | 常函数 | 1 | 是 |
| id(n) | 恒等函数 | n | 是 |
| id_k(n) | 幂函数 | n^k | 是 |
| μ(n) | 莫比乌斯函数 | 见 1.1 | 是 |
| φ(n) | 欧拉函数 | 1~n 中与 n 互质的个数 | 是 |
| σ_k(n) | 除数幂和 | Σ_{d|n} d^k | 是 |
| τ(n) | 约数个数 | Σ_{d|n} 1 | 是 |

所有上述函数都是**积性函数**（gcd(m,n)=1 ⇒ f(mn) = f(m)f(n)），因此可以用线性筛 O(n) 预处理。

## 核心问题与适用边界

### 何时需要莫比乌斯反演

莫比乌斯反演的核心应用场景是：**当你需要统计满足某条件的数对数量，且条件涉及 gcd、互质关系时**。

**容斥 vs 莫比乌斯**：

| 维度 | 传统容斥 | 莫比乌斯反演 |
|------|---------|-------------|
| 表达方式 | 根据质因子集合逐层加减 | 用 μ(d) 一步编码 |
| 适用范围 | 质因子较少时直观 | 任意范围均可自动表达 |
| 分块加速 | 难以结合 | 天然适合整除分块 |
| 渐近复杂度 | 依赖质因子个数 ω(n) | 可做到 O(√n) |

**典型信号**：看到形如 ΣΣ [gcd(i,j) = k] 或 Σ gcd(i,j) 的求和，立刻想到莫比乌斯。

### gcd 类求和转化为莫比乌斯求和的标准技巧

**标准三步法**：

**步骤 1**：设 f(k) = 满足 gcd(i,j) = k 的数对个数
**步骤 2**：设 F(k) = 满足 k | gcd(i,j) 的数对个数，即 k 整除 gcd

关键等式：
```
F(k) = Σ f(d)     且     F(k) = ⌊n/k⌋ × ⌊m/k⌋
       k|d
```

**步骤 3**：莫比乌斯反演得：
```
f(k) = Σ μ(d/k) × F(d) = Σ μ(d/k) × ⌊n/d⌋ × ⌊m/d⌋
       k|d               k|d
```

特别地，对于 f(1)：
```
f(1) = Σ μ(d) × ⌊n/d⌋ × ⌊m/d⌋
       d=1..min(n,m)
```

**常见变换模式**：

```
ΣΣ gcd(i,j)     = Σ d × f(1)  （每个 d 贡献 d 倍的内层 coprime 对）
                       n/d, m/d

ΣΣ [gcd(i,j)=k] = Σ μ(d) × ⌊n/(kd)⌋ × ⌊m/(kd)⌋

ΣΣ lcm(i,j)     = ΣΣ i×j / gcd(i,j)
                 = Σ (i×j) / d × [gcd(i,j)=d]
                 → 代换后转为莫比乌斯
```

### 分块加速的前提条件与边界

**整除分块（数论分块）**的核心观察：

对于给定正整数 n，函数 ⌊n/i⌋ 的取值只有 **O(√n)** 种。

**单调性**：当 i 从 1 增大到 n 时，⌊n/i⌋ 单调非增。
- i ≤ √n 时，取值从 n 下降到 √n，最多 √n 种
- i > √n 时，⌊n/i⌋ ≤ √n，最多 √n 种
- 总计：≤ 2√n 种取值

**块边界公式**：

给定左端点 l，右端点 r 为：
```
r = ⌊ n / ⌊n/l⌋ ⌋
```

即：⌊n/l⌋ = ⌊n/(l+1)⌋ = ... = ⌊n/r⌋，而 ⌊n/(r+1)⌋ < ⌊n/l⌋。

**适用莫比乌斯反演的条件**：

当求和式为 Σ μ(d) × f(⌊n/d⌋) × g(⌊m/d⌋) 时，若 μ(d) 的前缀和可以 O(1) 查询，则可以将 d 按 ⌊n/d⌋ 和 ⌊m/d⌋ 同时取常数的区间分块，每块内 μ(d) 求和乘以相同因子。

**边界**：
- 朴素求和 O(min(n,m)) 可接受时（n,m ≤ 10⁶）：直接枚举
- 需要分块优化时（n,m ≤ 10⁹）：必须预处理 μ 的前缀和
- 需要杜教筛时（n ≤ 10¹²）：用亚线性筛求 μ 前缀和

### 线性筛 O(n) 预处理 μ(n) vs O(√n) 单值求 μ(n)

**方案一：欧拉筛 O(n) 批量预处理**（推荐，当 n ≤ 10⁷ 时）

```java
int[] mobius(int n) {
    int[] mu = new int[n + 1];
    int[] primes = new int[n + 1];
    boolean[] isComp = new boolean[n + 1];
    int cnt = 0;
    mu[1] = 1;
    for (int i = 2; i <= n; i++) {
        if (!isComp[i]) {
            primes[cnt++] = i;
            mu[i] = -1;          // 素数：1 个质因子，μ = -1
        }
        for (int j = 0; j < cnt && i * primes[j] <= n; j++) {
            isComp[i * primes[j]] = true;
            if (i % primes[j] == 0) {
                mu[i * primes[j]] = 0;    // 平方因子出现 → μ = 0
                break;
            } else {
                mu[i * primes[j]] = -mu[i]; // 新增质因子，符号取反
            }
        }
    }
    return mu;
}
```

**欧拉筛递推的三种情形**：

| 条件 | mu[i×p] | 解释 |
|------|---------|------|
| i % p != 0，且 i 无平方因子 | -mu[i] | 增加一个不同质因子，符号翻转 |
| i % p != 0，且 i 有平方因子 | 0 | 新增因子无法消除平方因子 |
| i % p == 0 | 0 | i×p 中 p 出现至少两次，有平方因子 |

**方案二：质因数分解 O(√n) 单值求 μ(n)**（适用于单个大 n）

```java
// 单值求 μ(n)，O(√n)
int mobiusSingle(long n) {
    if (n == 1) return 1;
    int cnt = 0;
    for (long p = 2; p * p <= n; p++) {
        if (n % p == 0) {
            int exp = 0;
            while (n % p == 0) {
                n /= p;
                exp++;
            }
            if (exp >= 2) return 0;  // 平方因子
            cnt++;                    // 不同质因子计数
        }
    }
    if (n > 1) cnt++;                 // 剩余一个大质因子
    return (cnt % 2 == 0) ? 1 : -1;
}
```

## 高效实现与关键优化

### 欧拉筛预处理 μ(n) 的完整递推细节

欧拉筛的核心思想：每个合数只被其**最小质因子**筛掉一次，保证 O(n) 线性复杂度。

**对每个 i 的处理**：

```
对于当前 i，遍历已发现的质数 primes[j]：
  1. 标记 i × primes[j] 为合数
  2. 若 i % primes[j] == 0：
     a) i × primes[j] 的最小质因子就是 primes[j]
     b) i 已经包含 primes[j]，所以 i × primes[j] 含有 primes[j]²
     c) μ(i × primes[j]) = 0
     d) break（保证每个合数只被最小质因子筛一次）
  3. 否则（i % primes[j] != 0）：
     a) i × primes[j] 的最小质因子是 primes[j]
     b) 若 μ(i) ≠ 0（i 无平方因子），则 i×p 无平方因子，新增一个不同质因子
     c) μ(i × primes[j]) = -μ(i)
     d) 继续遍历下一个质数
```

**正确性证明**：

设合数 n = p₁^{a₁}p₂^{a₂}...pₖ^{aₖ}，其中 p₁ < p₂ < ... < pₖ。

欧拉筛中，n 被遍历到当 i = p₁^{a₁-1}p₂^{a₂}...pₖ^{aₖ} 且 primes[j] = p₁ 时。

此时：
- i 不含因子 p₁² 的前提是 a₁ - 1 ≤ 1，即 a₁ ∈ {1, 2}
  - a₁ = 1：i 无因子 p₁，i % p₁ ≠ 0，μ(i×p₁) = -μ(i)，正确
  - a₁ ≥ 2：i 含 p₁^{a₁-1}，i % p₁ = 0，μ(i×p₁) = 0，正确
- 无论如何，break 保证「每个合数只被筛一次」成立

### 前缀和预处理与整除分块

**前缀和数组**：

为了在分块中 O(1) 获取 μ 在区间 [l, r] 上的和，预处理：
```java
int[] prefixMu = new int[n + 1];
for (int i = 1; i <= n; i++) {
    prefixMu[i] = prefixMu[i - 1] + mu[i];
}
// 区间 [l, r] 的和 = prefixMu[r] - prefixMu[l - 1]
```

### 整除分块的原理与证明

**定理**：⌊n/i⌋ 的取值只有 O(√n) 种。

**块边界公式证明**：

给定 l，要找到最大的 r 使得 ⌊n/r⌋ = ⌊n/l⌋。

由定义，⌊n/r⌋ = ⌊n/l⌋ 等价于：
```
n/r ≥ ⌊n/l⌋ 且 n/(r+1) < ⌊n/l⌋
```

由 n/r ≥ ⌊n/l⌋ ⇒ r ≤ n/⌊n/l⌋，取整数：
```
r = ⌊ n / ⌊n/l⌋ ⌋
```

验证：对于任意 i ∈ [l, r]，⌊n/i⌋ 都等于 ⌊n/l⌋。

**双变量分块**：

当求和涉及 ⌊n/d⌋ 和 ⌊m/d⌋ 两个取整时，取两端 min：
```java
int r = (int)Math.min(n / (n / l), m / (m / l));
```

**复杂度证明**：

对 n 分块 O(√n)，对 m 分块 O(√m)，取 min 后仍为 O(√min(n,m))，而两个约束使块数 ≤ 2√n + 2√m = O(√n)，分块后的内层计算仍然是 O(1)，因此总复杂度 O(√n)。

### 积性筛预处理 φ, μ 的前缀和

由于 φ 和 μ 都是积性函数，可以在一次欧拉筛中同时预处理：

```java
// 同时预处理 mu 和 phi
class MobiusEulerSieve {
    int[] mu, phi, primes;
    boolean[] isComp;
    int cnt;

    MobiusEulerSieve(int n) {
        mu = new int[n + 1];
        phi = new int[n + 1];
        primes = new int[n + 1];
        isComp = new boolean[n + 1];
        cnt = 0;

        mu[1] = 1;
        phi[1] = 1;

        for (int i = 2; i <= n; i++) {
            if (!isComp[i]) {
                primes[cnt++] = i;
                mu[i] = -1;        // 素数：-1
                phi[i] = i - 1;    // 素数 p：φ(p) = p-1
            }
            for (int j = 0; j < cnt && i * primes[j] <= n; j++) {
                int p = primes[j];
                int ip = i * p;
                isComp[ip] = true;

                if (i % p == 0) {
                    // i 已被 p 整除
                    mu[ip] = 0;                // p² | ip
                    phi[ip] = phi[i] * p;      // p | i，φ(ip) = φ(i) × p
                    break;
                } else {
                    // i 与 p 互质
                    mu[ip] = -mu[i];            // 新增一个质因子
                    phi[ip] = phi[i] * (p - 1); // 积性函数性质：φ(ab) = φ(a)×φ(b)（a⊥b）
                }
            }
        }
    }

    // 前缀和
    long[] prefixMu() {
        long[] pre = new long[mu.length];
        for (int i = 1; i < mu.length; i++) {
            pre[i] = pre[i - 1] + mu[i];
        }
        return pre;
    }

    long[] prefixPhi() {
        long[] pre = new long[phi.length];
        for (int i = 1; i < phi.length; i++) {
            pre[i] = pre[i - 1] + phi[i];
        }
        return pre;
    }
}
```

**筛选 φ 的递推公式**：

| 条件 | φ(i×p) | 推导 |
|------|--------|------|
| i % p != 0，i 与 p 互质 | φ(i) × (p-1) | 积性函数性质 |
| i % p == 0，p | i | φ(i) × p | φ(p^{k+1}) = φ(p^k) × p |

### 杜教筛的基本思想简介

**问题**：求积性函数 f 的前缀和 S(n) = Σ_{i=1}^{n} f(i)，其中 n 很大（如 n ≤ 10¹²）。

O(n) 线性筛在 n > 10⁷ 时不可行，此时需要亚线性算法——杜教筛。

**核心公式**：

对于积性函数 f，构造函数 g 使得 h = f * g 的前缀和容易计算。则：

```
g(1)S(n) = Σ h(i) - Σ g(d) × S(⌊n/d⌋)
            i=1..n      d=2..n
```

其中第二项可以通过整除分块和递归/记忆化计算。

**推导**：

```
由 h = f * g：
h(n) = Σ f(d) × g(n/d)
       d|n

两侧求前缀和：
Σ h(i) = Σ Σ f(d) × g(i/d)
 i=1..n   i=1..n d|i

交换求和顺序，令 i = d × k：
= Σ g(k) × Σ f(d)
  k=1..n      d ≤ n/k

= Σ g(k) × S(⌊n/k⌋)
  k=1..n

= g(1) × S(n) + Σ g(k) × S(⌊n/k⌋)
                 k=2..n

移项：
g(1)S(n) = Σ_{i=1}^{n} h(i) - Σ_{k=2}^{n} g(k) × S(⌊n/k⌋)
```

**常用配对**：

| f | g | h = f * g | 应用场景 |
|---|---|-----------|---------|
| μ | 1 | ε | 求 μ 前缀和 |
| φ | 1 | id | 求 φ 前缀和 |
| id_k·μ | id_k | ε | 带权 μ 前缀和 |

**以 μ 为例**：

取 g = 1（常函数），则 h = μ * 1 = ε，Σ_{i=1}^{n} ε(i) = 1。

```
1 × S_μ(n) = 1 - Σ S_μ(⌊n/k⌋)
                 k=2..n
```

递归 + 整除分块 + 记忆化 → O(n^{2/3}) 可处理 n ≤ 10¹²。

## 典型题目与解题思路

### ΣΣ gcd(i,j) —— 标准莫比乌斯变换

**问题**：给定 n, m，求 Σ_{i=1}^{n} Σ_{j=1}^{m} gcd(i, j)

**推导**：

```
Ans = Σ Σ gcd(i, j) = Σ d × [gcd(i, j) = d] 的数对个数
     i=1..n j=1..m     d=1..min(n,m)

设 f(k) = gcd(i,j) = k 的数对个数（在 n,m 限制下）

f(k) = Σ μ(t) × ⌊n/(kt)⌋ × ⌊m/(kt)⌋
       t=1..min(n,m)/k

Ans = Σ d × Σ μ(t) × ⌊n/(dt)⌋ × ⌊m/(dt)⌋
      d       t

交换求和，令 T = d × t：

Ans = Σ ⌊n/T⌋ × ⌊m/T⌋ × Σ (d × μ(T/d))
      T=1..min(n,m)           d|T

令 F(T) = Σ d × μ(T/d) = (id * μ)(T) = φ(T)  （经典卷积：id * μ = φ）
           d|T

因此：Ans = Σ ⌊n/T⌋ × ⌊m/T⌋ × φ(T)
           T=1..min(n,m)
```

这是一个非常优美的结论：**gcd 求和约化到了 φ 函数**。

**完整 Java 实现**：

```java
import java.util.*;

public class SumGcd {
    int[] phi;
    long[] prefixPhi;

    // 欧拉筛 O(n)
    void preprocess(int n) {
        phi = new int[n + 1];
        int[] primes = new int[n + 1];
        boolean[] isComp = new boolean[n + 1];
        int cnt = 0;
        phi[1] = 1;

        for (int i = 2; i <= n; i++) {
            if (!isComp[i]) {
                primes[cnt++] = i;
                phi[i] = i - 1;
            }
            for (int j = 0; j < cnt && i * primes[j] <= n; j++) {
                int p = primes[j];
                int ip = i * p;
                isComp[ip] = true;
                if (i % p == 0) {
                    phi[ip] = phi[i] * p;
                    break;
                } else {
                    phi[ip] = phi[i] * (p - 1);
                }
            }
        }

        prefixPhi = new long[n + 1];
        for (int i = 1; i <= n; i++) {
            prefixPhi[i] = prefixPhi[i - 1] + phi[i];
        }
    }

    // 朴素 O(min(n,m))
    long solveNaive(int n, int m) {
        preprocess(Math.min(n, m));
        int N = Math.min(n, m);
        long ans = 0;
        for (int T = 1; T <= N; T++) {
            ans += (long)(n / T) * (m / T) * phi[T];
        }
        return ans;
    }

    // 整除分块 O(√n)
    long solveWithBlock(int n, int m) {
        preprocess(Math.min(n, m));
        int N = Math.min(n, m);
        long ans = 0;
        for (int l = 1, r; l <= N; l = r + 1) {
            int nDiv = n / l, mDiv = m / l;
            r = Math.min(n / nDiv, m / mDiv);
            r = Math.min(r, N);
            ans += (long) nDiv * mDiv * (prefixPhi[r] - prefixPhi[l - 1]);
        }
        return ans;
    }

    public static void main(String[] args) {
        SumGcd sg = new SumGcd();
        // 测试用例：n=m=5
        // gcd矩阵：
        // 1 1 1 1 1
        // 1 2 1 2 1
        // 1 1 3 1 1
        // 1 2 1 4 1
        // 1 1 1 1 5
        // 求和 = 1*9 + 2*6 + 3*2 + 4*1 + 5*1 = 9+12+6+4+5 = 36
        System.out.println("n=m=5: " + sg.solveNaive(5, 5));         // 36
        System.out.println("n=m=5: " + sg.solveWithBlock(5, 5));    // 36
        System.out.println("n=m=100: " + sg.solveWithBlock(100, 100));
        System.out.println("n=1000,m=2000: " + sg.solveWithBlock(1000, 2000));
    }
}
```

**复杂度**：
- 预处理：O(N) 时间，O(N) 空间
- 朴素求值：O(min(n,m))
- 分块求值：O(√min(n,m))
- 测试：n=m=5 → ans=36 ✓

### 互质对计数 ΣΣ [gcd(i,j)=1]

**问题**：给定 n, m，求满足 1 ≤ i ≤ n, 1 ≤ j ≤ m 且 gcd(i, j) = 1 的数对个数。

**推导**：

```
Ans = Σ Σ [gcd(i, j) = 1]
     i=1..n j=1..m

由核心恒等式 [n=1] = Σ μ(d)：
                      d|n

[gcd(i,j)=1] = Σ μ(d)
                d|gcd(i,j)

             = Σ μ(d) × [d|i] × [d|j]
               d=1..min(n,m)

交换求和顺序：
Ans = Σ μ(d) × (满足 d|i 的 i 个数) × (满足 d|j 的 j 个数)
     d=1..min(n,m)

   = Σ μ(d) × ⌊n/d⌋ × ⌊m/d⌋
     d=1..min(n,m)
```

**完整 Java 实现**：

```java
                    mu[ip] = 0;
                    break;

                }
            }
        }

        for (int d = 1; d <= N; d++) {

        // (1,1)(1,2)(1,3)(2,1)(2,3)(3,1)(3,2) → 7 对
        // (2,2) gcd=2, (3,3) gcd=3 排除
        System.out.println("n=m=3: " + cp.countWithBlock(3, 3)); // 7

        // 测试用例 3：n=5,m=5 手动计算验证
        // 互质对: 除 (2,2)(2,4)(3,3)(4,2)(4,4)(5,5) 外其余均为互质？
        // 全部 25 对，不互质的：2,2 2,4 3,3 4,2 4,4 5,5 → 6 对
        // 25-6 = 19 对？ 实际上还有(2,6)之类 n=5,m=5 范围内仅以上
        // 再检查：(4,2) gcd=2, (4,4) gcd=4 但是 4,4 超出？n=5,m=5 范围内
        // (2,4) gcd=2, (4,2) gcd=2, (4,4) gcd=4, (5,5) gcd=5
        // 还剩 25-4 = 21? 不对，仔细列出：
        // gcd=1 的：除 (2,2)(2,4)(3,3)(4,2)(4,4)(5,5) 外全部
        // 所以是 25-6 = 19
        // (1,1)~(5,5)全量验证略，这里用公式验证：
        long ans = cp.countWithBlock(5, 5);
        System.out.println("n=m=5: " + ans); // 期望值

        // 测试用例 4：n=10, m=10
        System.out.println("n=m=10: " + cp.countWithBlock(10, 10));

        // 测试用例 5：大规模，验证分块正确性
        System.out.println("n=10000,m=10000: " + cp.countWithBlock(10000, 10000));
        System.out.println("朴素结果: " + cp.countNaive(10000, 10000));
    }
}
```

- 询问：O(√min(n,m))
- 关键点：n/d 和 m/d 同时分块使块数 ≤ 2√n + 2√m

### 最大公约数为 k 的数对

**问题**：给定 n, m, k，求满足 gcd(i, j) = k 的数对个数（1 ≤ i ≤ n, 1 ≤ j ≤ m）。

**推导**：

```
gcd(i, j) = k  ⇔  k | i, k | j 且 gcd(i/k, j/k) = 1

因此：
令 i' = i/k, j' = j/k，则 1 ≤ i' ≤ ⌊n/k⌋, 1 ≤ j' ≤ ⌊m/k⌋

Ans = 在 ⌊n/k⌋, ⌊m/k⌋ 范围内互质对的个数
    = Σ μ(d) × ⌊n/(kd)⌋ × ⌊m/(kd)⌋
      d=1..min(⌊n/k⌋, ⌊m/k⌋)
```

**完整 Java 实现**：

```java
import java.util.*;

public class GcdEqualsK {
    int[] mu;
    long[] prefixMu;

    void preprocess(int n) {
        mu = new int[n + 1];
        prefixMu = new long[n + 1];
        int[] primes = new int[n + 1];
        boolean[] isComp = new boolean[n + 1];
        int cnt = 0;
        mu[1] = 1;
        for (int i = 2; i <= n; i++) {
            if (!isComp[i]) {
                primes[cnt++] = i;
                mu[i] = -1;
            }
            for (int j = 0; j < cnt && i * primes[j] <= n; j++) {
                int p = primes[j];
                int ip = i * p;
                isComp[ip] = true;
                if (i % p == 0) {
                    mu[ip] = 0;
                    break;
                } else {
                    mu[ip] = -mu[i];
                }
            }
        }
        for (int i = 1; i <= n; i++) {
            prefixMu[i] = prefixMu[i - 1] + mu[i];
        }
    }

    // 求 gcd(i,j) = k 的对数
    long countGcdK(int n, int m, int k) {
        if (k > n || k > m) return 0;
        int N = Math.min(n / k, m / k);
        if (N <= 0) return 0;
        preprocess(N);
        long ans = 0;
        for (int l = 1, r; l <= N; l = r + 1) {
            int nd = (n / k) / l;
            int md = (m / k) / l;
            if (nd == 0 || md == 0) break;
            r = Math.min((n / k) / nd, (m / k) / md);
            r = Math.min(r, N);
            ans += (long) nd * md * (prefixMu[r] - prefixMu[l - 1]);
        }
        return ans;
    }

    // 扩展：求所有 gcd 的分布
    long[] countAllGcd(int n, int m) {
        int N = Math.min(n, m);
        long[] ans = new long[N + 1];
        // 反向推导：gcd = k 的对数
        for (int k = N; k >= 1; k--) {
            long total = (long)(n / k) * (m / k);
            long exclude = 0;
            for (int t = 2; t * k <= N; t++) {
                exclude += ans[t * k];
            }
            ans[k] = total - exclude;
        }
        return ans;
    }

    public static void main(String[] args) {
        GcdEqualsK gk = new GcdEqualsK();

        // 测试用例 1：n=6,m=6,k=2
        // gcd=2 的数对：(2,2)(2,4)(2,6)(4,2)(4,6)(6,2)(6,4)(6,6)
        // 但注意 (4,2) gcd=2, (6,4) gcd=2... 等等
        // 等价于在 n/k=3,m/k=3 范围内互质对个数
        System.out.println("n=6,m=6,k=2: " + gk.countGcdK(6, 6, 2));

        // 测试用例 2：n=5,m=5,k=1
        System.out.println("n=5,m=5,k=1: " + gk.countGcdK(5, 5, 1));

        // 测试用例 3：n=10,m=10,k=3
        // n/k=3,m/k=3 互质对数
        System.out.println("n=10,m=10,k=3: " + gk.countGcdK(10, 10, 3));

        // 测试用例 4：所有 gcd 分布
        long[] dist = gk.countAllGcd(6, 6);
        System.out.print("gcd分布 [1..6]: ");
        for (int i = 1; i < dist.length; i++) {
            System.out.print("gcd=" + i + ":" + dist[i] + " ");
        }
        System.out.println();
        // 验证：所有分布之和 = 36
        long sum = 0;
        for (int i = 1; i < dist.length; i++) sum += dist[i];
        System.out.println("分布之和=" + sum + " (应为36)");
    }
}
```

**复杂度**：
- 单次询问：O(√min(n/k, m/k))
- 全分布：O(N log N)
- 空间：O(N)

### lcm 求和 ΣΣ lcm(i,j)

**问题**：给定 n, m，求 Σ_{i=1}^{n} Σ_{j=1}^{m} lcm(i, j)

**推导**：

```
lcm(i,j) = i × j / gcd(i,j)

Ans = Σ Σ i × j / gcd(i, j)

枚举 gcd = d：
= Σ Σ (i×j/d) × [gcd(i,j)=d]
  d   i,j

令 i = d×i', j = d×j'，gcd(i',j')=1：

= Σ Σ (d² × i' × j') / d × [gcd(i',j')=1]
  d   i'≤n/d, j'≤m/d

= Σ d × Σ Σ i' × j' × [gcd(i',j')=1]
  d       i'≤⌊n/d⌋  j'≤⌊m/d⌋

令 S(n,m) = Σ Σ i × j × [gcd(i,j)=1]
            i≤n  j≤m

则 Ans = Σ d × S(⌊n/d⌋, ⌊m/d⌋)
          d=1..min(n,m)

下面求 S(n,m)：

S(n,m) = Σ Σ i × j × Σ μ(t)
          i≤n  j≤m      t|gcd(i,j)

       = Σ μ(t) × (满足 t|i 的 i 的个数和) × (满足 t|j 的 j 的个数和)

       = Σ μ(t) × t² × (1+2+...+⌊n/t⌋) × (1+2+...+⌊m/t⌋)

       = Σ μ(t) × t² × (⌊n/t⌋(⌊n/t⌋+1)/2) × (⌊m/t⌋(⌊m/t⌋+1)/2)
         t=1..min(n,m)
```

这是双层嵌套结构，外层枚举 d，内层枚举 t，都可以用整除分块优化。

**完整 Java 实现**：

```java
import java.util.*;

public class SumLcm {
    int[] mu;
    long[] prefixMu2; // 前缀和：Σ μ(t) × t²

    static final long MOD = 1000000007;

    void preprocess(int n) {
        mu = new int[n + 1];
        int[] primes = new int[n + 1];
        boolean[] isComp = new boolean[n + 1];
        int cnt = 0;
        mu[1] = 1;
        for (int i = 2; i <= n; i++) {
            if (!isComp[i]) {
                primes[cnt++] = i;
                mu[i] = -1;
            }
            for (int j = 0; j < cnt && i * primes[j] <= n; j++) {
                int p = primes[j];
                int ip = i * p;
                isComp[ip] = true;
                if (i % p == 0) {
                    mu[ip] = 0;
                    break;
                } else {
                    mu[ip] = -mu[i];
                }
            }
        }
        prefixMu2 = new long[n + 1];
        for (int i = 1; i <= n; i++) {
            long val = (long) mu[i] * i % MOD * i % MOD;
            prefixMu2[i] = (prefixMu2[i - 1] + val + MOD) % MOD;
        }
    }

    // 三角形数：1+2+...+x = x(x+1)/2
    long tri(long x) {
        return x * (x + 1) / 2 % MOD;
    }

    // 计算 S(n,m) = ΣΣ i×j×[gcd(i,j)=1]
    long S(int n, int m) {
        if (n <= 0 || m <= 0) return 0;
        int N = Math.min(n, m);
        long ans = 0;
        for (int l = 1, r; l <= N; l = r + 1) {
            int nd = n / l, md = m / l;
            r = Math.min(n / nd, m / md);
            r = Math.min(r, N);
            long mu2Sum = (prefixMu2[r] - prefixMu2[l - 1] + MOD) % MOD;
            long part = tri(nd) % MOD * tri(md) % MOD;
            ans = (ans + mu2Sum * part) % MOD;
        }
        return ans;
    }

    // 计算 ΣΣ lcm(i,j)
    long solve(int n, int m) {
        preprocess(Math.min(n, m));
        int N = Math.min(n, m);
        long ans = 0;
        for (int l = 1, r; l <= N; l = r + 1) {
            int nd = n / l, md = m / l;
            r = Math.min(n / nd, m / md);
            r = Math.min(r, N);
            // 外层求和：Σ d × S(n/d, m/d)
            // d 在 [l, r] 范围上线性增长，这里朴素计算内层 S
            // 优化：此处为双层分块结构，更优方案见后文
            for (int d = l; d <= r; d++) {
                ans = (ans + (long) d * S(n / d, m / d)) % MOD;
            }
        }
        return ans;
    }

    public static void main(String[] args) {
        SumLcm sl = new SumLcm();

        // 测试用例 1：n=m=1
        // lcm(1,1)=1
        System.out.println("n=m=1: " + sl.solve(1, 1)); // 1

        // 测试用例 2：n=m=2
        // lcm(1,1)=1, lcm(1,2)=2, lcm(2,1)=2, lcm(2,2)=2
        // 总和 = 1+2+2+2 = 7
        System.out.println("n=m=2: " + sl.solve(2, 2)); // 7

        // 测试用例 3：n=m=3
        // 手动验证或依赖程序
        System.out.println("n=m=3: " + sl.solve(3, 3));

        // 测试用例 4：n=m=5
        System.out.println("n=m=5: " + sl.solve(5, 5));
    }
}
```

**双层分块优化**（进阶）：

当 n,m 很大时，内层 S(n/d, m/d) 对每个 d 枚举会退化。注意到 S 只依赖于 n/d 和 m/d 的值，而这些值与 d 的取值相关。可以用以下更高效的方法：

```
Ans = Σ d × Σ μ(t) × t² × tri(⌊n/(dt)⌋) × tri(⌊m/(dt)⌋)
      d       t

令 T = d × t，枚举 T：
= Σ tri(⌊n/T⌋) × tri(⌊m/T⌋) × Σ d × μ(T/d) × (T/d)²
  T                          d|T

= Σ tri(⌊n/T⌋) × tri(⌊m/T⌋) × T² × Σ (1/d) × μ(d)
  T                                       d|T

其中内层 Σ_{d|T} (1/d) × μ(d) 可以将 μ(d)/d 做狄利克雷卷积与 1 卷积后，与 T² 相乘。

这进一步化简后可用另一个积性函数（μ(d)/d 的卷积）加速，但一般竞赛中不要求做如此深的优化，多数题目 n,m ≤ 10⁵~10⁶ 时纯莫比乌斯 O(n) 即可通过。
```

**复杂度**：
- 朴素双层：O(n × √n) ≈ O(n^{1.5})
- 单层转换为直接枚举后加整除分块：O(n)
- 全分块优化：O(√n)

### 乘积互质对

```java
            int nd = n / l;
            int md = m / l;
            // nd 和 md 在区间 [l, r] 上不变
            r = Math.min(n / nd, m / md);
            r = Math.min(r, N);
            ans += (long) nd * md * (prefixMu[r] - prefixMu[l - 1]);
        }
        return ans;
    }

    // 朴素版本作为正确性验证
    long countCoprimeNaive(int n, int m) {
        int N = Math.min(n, m);
        preprocess(N);
        long ans = 0;
        for (int d = 1; d <= N; d++) {
            ans += (long) mu[d] * (n / d) * (m / d);
        }
        return ans;
    }

    public static void main(String[] args) {
        ProductCoprime pc = new ProductCoprime();

        // 边界测试
        System.out.println("=== 边界测试 ===");
        System.out.println("n=1,m=1: " + pc.countCoprime(1, 1));   // 1
        System.out.println("n=1,m=10: " + pc.countCoprime(1, 10)); // 10
        System.out.println("n=0,m=0: " + pc.countCoprime(0, 0));   // 0

        // 正确性测试：分块 vs 朴素
        System.out.println("\n=== 正确性验证 ===");
        int[] testNs = {10, 50, 100, 500, 1000, 5000};
        for (int n : testNs) {
            long a = pc.countCoprime(n, n);
            long b = pc.countCoprimeNaive(n, n);
            System.out.println("n=" + n + "  block=" + a + " naive=" + b + " " + (a == b ? "✓" : "✗"));
        }

        // 性能测试
        System.out.println("\n=== 性能测试 ===");
        long start = System.currentTimeMillis();
        long ans = pc.countCoprime(100000, 100000);
        long end = System.currentTimeMillis();
        System.out.println("n=m=100k, ans=" + ans + ", time=" + (end - start) + "ms");

        start = System.currentTimeMillis();
        ans = pc.countCoprimeNaive(100000, 100000);
        end = System.currentTimeMillis();
        System.out.println("n=m=100k(naive), ans=" + ans + ", time=" + (end - start) + "ms");

        // 不对称情况
        System.out.println("n=1000,m=5000: " + pc.countCoprime(1000, 5000));
        System.out.println("n=5000,m=1000: " + pc.countCoprime(5000, 1000));
        // 对称性验证
        System.out.println("对称验证: " + (pc.countCoprime(1000,5000) == pc.countCoprime(5000,1000) ? "✓" : "✗"));
    }
}
```

- 分块版本：O(√min(n,m))
- 朴素版本：O(min(n,m))
- n=100k 时分块约比朴素快 300 倍以上

### 区间互质对（L, R）

**问题**：给定区间 [L₁, R₁] 和 [L₂, R₂]，求满足 L₁ ≤ i ≤ R₁, L₂ ≤ j ≤ R₂ 且 gcd(i,j) = 1 的数对个数。

**推导**：

利用二维前缀和思想，定义：
```
F(n,m) = Σ_{i=1}^{n} Σ_{j=1}^{m} [gcd(i,j) = 1]
```

则区间答案为：
```
Ans = F(R₁, R₂) - F(L₁-1, R₂) - F(R₁, L₂-1) + F(L₁-1, L₂-1)
```

这就是二维前缀和的容斥，将区间计数转化为 4 次前缀计数。

**完整 Java 实现**：

```java
import java.util.*;

public class RangeCoprime {
    int[] mu;
    long[] prefixMu;

    void preprocess(int n) {
        mu = new int[n + 1];
        prefixMu = new long[n + 1];
        int[] primes = new int[n + 1];
        boolean[] isComp = new boolean[n + 1];
        int cnt = 0;
        mu[1] = 1;
        for (int i = 2; i <= n; i++) {
            if (!isComp[i]) {
                primes[cnt++] = i;
                mu[i] = -1;
            }
            for (int j = 0; j < cnt && i * primes[j] <= n; j++) {
                int p = primes[j];
                int ip = i * p;
                isComp[ip] = true;
                if (i % p == 0) {
                    mu[ip] = 0;
                    break;
                } else {
                    mu[ip] = -mu[i];
                }
            }
        }
        for (int i = 1; i <= n; i++) {
            prefixMu[i] = prefixMu[i - 1] + mu[i];
        }
    }

    // 计算 [1..n] × [1..m] 互质对个数（分块版本）
    long F(int n, int m) {
        if (n <= 0 || m <= 0) return 0;
        int N = Math.min(n, m);
        preprocess(N);
        long ans = 0;
        for (int l = 1, r; l <= N; l = r + 1) {
            int nd = n / l, md = m / l;
            r = Math.min(n / nd, m / md);
            r = Math.min(r, N);
            ans += (long) nd * md * (prefixMu[r] - prefixMu[l - 1]);
        }
        return ans;
    }

    // 区间互质对：[L1,R1] × [L2,R2]
    long rangeCoprime(int L1, int R1, int L2, int R2) {
        return F(R1, R2) - F(L1 - 1, R2) - F(R1, L2 - 1) + F(L1 - 1, L2 - 1);
    }

    // 朴素算法用于验证
    long rangeCoprimeNaive(int L1, int R1, int L2, int R2) {
        long ans = 0;
        for (int i = L1; i <= R1; i++) {
            for (int j = L2; j <= R2; j++) {
                if (gcd(i, j) == 1) ans++;
            }
        }
        return ans;
    }

    int gcd(int a, int b) {
        return b == 0 ? a : gcd(b, a % b);
    }

    public static void main(String[] args) {
        RangeCoprime rc = new RangeCoprime();

        // 测试用例 1：[1,3] × [1,3]
        long ans1 = rc.rangeCoprime(1, 3, 1, 3);
        long ans1b = rc.rangeCoprimeNaive(1, 3, 1, 3);
        System.out.println("[1,3]×[1,3]: " + ans1 + " (naive=" + ans1b + ") " + (ans1 == ans1b ? "✓" : "✗"));

        // 测试用例 2：[2,5] × [3,7]
        long ans2 = rc.rangeCoprime(2, 5, 3, 7);
        long ans2b = rc.rangeCoprimeNaive(2, 5, 3, 7);
        System.out.println("[2,5]×[3,7]: " + ans2 + " (naive=" + ans2b + ") " + (ans2 == ans2b ? "✓" : "✗"));

        // 测试用例 3：[1,10] × [1,10]
        long ans3 = rc.rangeCoprime(1, 10, 1, 10);
        long ans3b = rc.rangeCoprimeNaive(1, 10, 1, 10);
        System.out.println("[1,10]×[1,10]: " + ans3 + " (naive=" + ans3b + ") " + (ans3 == ans3b ? "✓" : "✗"));

        // 测试用例 4：对称性验证
        System.out.println("对称性: " + rc.rangeCoprime(2,7,3,8) + " vs " +
            rc.rangeCoprime(3,8,2,7) + " " +
            (rc.rangeCoprime(2,7,3,8) == rc.rangeCoprime(3,8,2,7) ? "✓" : "✗"));

        // 测试用例 5：大区间但不依赖朴素验证
        System.out.println("[1,100000]×[1,100000]: " + rc.rangeCoprime(1, 100000, 1, 100000));
    }
}
```

**复杂度**：
- 每次 F(n,m) 调用：O(√min(n,m))
- 区间询问：4 次 F 调用，O(√max(R₁,R₂))
- 注意：每次预处理需要到当前最大范围，若多次询问可在全局预处理后用

### 除数函数求和：∑ σ_k(i)

**问题**：求 S_k(n) = Σ_{i=1}^{n} σ_k(i)，其中 σ_k(n) = Σ_{d|n} d^k（所有正因子的 k 次幂之和）。

**推导**：

```
σ_k(n) = Σ d^k
         d|n

S_k(n) = Σ σ_k(i)
         i=1..n

       = Σ Σ d^k
         i=1..n d|i

交换求和顺序——枚举除数 d 的贡献：
对于每个 d，在 1..n 内有多少个 i 是 d 的倍数？恰好 ⌊n/d⌋ 个。

S_k(n) = Σ d^k × ⌊n/d⌋
         d=1..n
```

这就是除数函数前缀和的经典公式，简洁且可以整除分块计算。

**特例**：
- k = 0：σ₀(n) = τ(n)（约数个数），S₀(n) = Σ_{i=1}^{n} τ(i) = Σ_{d=1}^{n} ⌊n/d⌋
- k = 1：σ₁(n) = σ(n)（约数和），S₁(n) = Σ_{d=1}^{n} d × ⌊n/d⌋

**完整 Java 实现**：

```java
import java.util.*;

public class DivisorSum {
    static final long MOD = 1000000007;

    // 快速幂
    long powMod(long a, int k, long mod) {
        long res = 1 % mod;
        while (k > 0) {
            if ((k & 1) == 1) res = res * a % mod;
            a = a * a % mod;
            k >>= 1;
        }
        return res;
    }

    // 求 Σ_{i=1}^{n} i^k
    // 对于小 k，直接用公式。这里用 O(k log k) 求和。
    // 注意：大 n 时 i^k 可能很大，这里只是演示思路
    long sumPow(long n, int k, long mod) {
        long ans = 0;
        for (long i = 1; i <= n; i++) {
            ans = (ans + powMod(i, k, mod)) % mod;
        }
        return ans;
    }

    // 直接公式 S_k(n) = Σ d^k × ⌊n/d⌋, O(n)
    long solveNaive(long n, int k) {
        long ans = 0;
        for (long d = 1; d <= n; d++) {
            ans = (ans + powMod(d, k, MOD) * (n / d)) % MOD;
        }
        return ans;
    }

    // 整除分块版本 O(√n)
    long solveWithBlock(long n, int k) {
        long ans = 0;
        for (long l = 1, r; l <= n; l = r + 1) {
            long val = n / l;          // ⌊n/l⌋
            r = n / val;                // 块右端点
            // 需要 Σ_{i=l}^{r} i^k
            // 实际竞赛中，由于 k 较小（通常 k=0,1,2），可以 O(1) 求 r 次幂和
            long sumK = 0;
            for (long i = l; i <= r; i++) {
                sumK = (sumK + powMod(i, k, MOD)) % MOD;
            }
            ans = (ans + sumK * val) % MOD;
        }
        return ans;
    }

    // 更高效的实现：对于 k=1,2 等小幂次，用公式
    long sumPow1(long n) { // 1+2+...+n = n(n+1)/2
        return n % MOD * ((n + 1) % MOD) % MOD * ((MOD + 1) / 2) % MOD;
    }

    long sumPow2(long n) { // 1²+2²+...+n² = n(n+1)(2n+1)/6
        long a = n % MOD;
        long b = (n + 1) % MOD;
        long c = (2 * n + 1) % MOD;
        long inv6 = (MOD + 1) / 6; // 6 的模逆
        return a * b % MOD * c % MOD * inv6 % MOD;
    }

    // 特化版 k=1 的分块计算
    long solveK1(long n) {
        long ans = 0;
        for (long l = 1, r; l <= n; l = r + 1) {
            long val = n / l;
            r = n / val;
            // Σ_{i=l}^{r} i = (l+r)(r-l+1)/2
            long sumI = (l + r) % MOD * ((r - l + 1) % MOD) % MOD * ((MOD + 1) / 2) % MOD;
            ans = (ans + sumI * (val % MOD)) % MOD;
        }
        return ans;
    }

    public static void main(String[] args) {
        DivisorSum ds = new DivisorSum();

        // 测试用例 1：k=0 (约数个数前缀和), n=5
        // σ₀(1)=1, σ₀(2)=2, σ₀(3)=2, σ₀(4)=3, σ₀(5)=2 → S=10
        // 公式：Σ⌊5/d⌋ = 5+2+1+1+1 = 10
        System.out.println("S₀(5)=" + ds.solveNaive(5, 0)); // 10

        // 测试用例 2：k=1 (约数和前缀和), n=5
        // σ₁(1)=1, σ₁(2)=1+2=3, σ₁(3)=1+3=4, σ₁(4)=1+2+4=7, σ₁(5)=1+5=6
        // S=1+3+4+7+6=21
        System.out.println("S₁(5)=" + ds.solveNaive(5, 1)); // 21
        System.out.println("S₁(5) block=" + ds.solveK1(5));   // 21

        // 测试用例 3：k=2, n=5
        // σ₂(1)=1¹=... 不对，σ₂(n)=Σ d²
        // σ₂(1)=1, σ₂(2)=1+4=5, σ₂(3)=1+9=10, σ₂(4)=1+4+16=21, σ₂(5)=1+25=26
        // S=1+5+10+21+26=63
        // 公式：Σ d²×⌊5/d⌋ = 1·5 + 4·2 + 9·1 + 16·1 + 25·1 = 5+8+9+16+25 = 63
        System.out.println("S₂(5)=" + ds.solveNaive(5, 2)); // 63

        // 测试用例 4：分块 vs 朴素
        System.out.println("S₁(1000) naive=" + ds.solveNaive(1000, 1)
            + " block=" + ds.solveK1(1000));

        // 测试用例 5：大规模 k=1
        long start = System.currentTimeMillis();
        long ans = ds.solveK1(1000000000L); // 10⁹
        long end = System.currentTimeMillis();
        System.out.println("S₁(10⁹)=" + ans + " time=" + (end - start) + "ms");
    }
}
```

**公式总结**：

```
S_k(n) = Σ_{d=1}^{n} d^k × ⌊n/d⌋

特殊：
S₀(n) = Σ_{d=1}^{n} ⌊n/d⌋ ≈ n log n  （调和级数）
S₁(n) = Σ_{d=1}^{n} d × ⌊n/d⌋         （可用分块 O(√n)）
```

**复杂度**：
- k=0 整除分块 O(√n)
- k=1 整除分块 + 等差数列求和 O(√n)
- 一般 k 需要 O(k√n) 或更强的幂和公式

### 莫比乌斯反演 + 杜教筛：求 φ 前缀和（n ≤ 10¹²）

**问题**：求 S(n) = Σ_{i=1}^{n} φ(i)，其中 n ≤ 10¹²，线性筛无法处理。

**杜教筛推导**：

目标函数：f = φ
构造函数：g = 1（常函数）
卷积：h = f * g = φ * 1 = id（核心性质：Σ_{d|n} φ(d) = n）

代入杜教筛公式：

```
g(1)S(n) = Σ h(i) - Σ g(d) × S(⌊n/d⌋)
           i=1..n     d=2..n

即：
1 × S_φ(n) = Σ_{i=1}^{n} i - Σ_{d=2}^{n} S_φ(⌊n/d⌋)

S_φ(n)         = n(n+1)/2 - Σ_{d=2}^{n} S_φ(⌊n/d⌋)
```

右侧的 Σ_{d=2}^{n} S_φ(⌊n/d⌋) 可以用整除分块计算，每个 ⌊n/d⌋ 只有 O(√n) 种取值。递归下去，总复杂度 O(n^{2/3})。

**完整 Java 实现**：

```java
import java.util.*;

public class DuJiaoSieve {
    // 预处理部分（n ≤ Nmax = 10⁶ 时用线性筛）
    static final int NMAX = 5000000;       // n^{2/3} 约 10⁶ 级别的平衡点
    long[] phi;                             // phi 前缀和，线性筛部分
    Map<Long, Long> memo;                  // 记忆化已计算的 S(n)

    DuJiaoSieve() {
        phi = new long[NMAX + 1];
        memo = new HashMap<>();
        linearSieve(NMAX);
    }

    // 线性筛 φ
    void linearSieve(int n) {
        int[] primes = new int[n + 1];
        boolean[] isComp = new boolean[n + 1];
        int cnt = 0;
        // phiVal 存 φ(i) 本身
        long[] phiVal = new long[n + 1];
        phiVal[1] = 1;
        for (int i = 2; i <= n; i++) {
            if (!isComp[i]) {
                primes[cnt++] = i;
                phiVal[i] = i - 1;
            }
            for (int j = 0; j < cnt && i * primes[j] <= n; j++) {
                int p = primes[j];
                int ip = i * p;
                isComp[ip] = true;
                if (i % p == 0) {
                    phiVal[ip] = phiVal[i] * p;
                    break;
                } else {
                    phiVal[ip] = phiVal[i] * (p - 1);
                }
            }
        }
        // 前缀和
        phi[0] = 0;
        for (int i = 1; i <= n; i++) {
            phi[i] = phi[i - 1] + phiVal[i];
        }
    }

    // 计算 S(n) = Σ φ(i)
    long S(long n) {
        if (n <= NMAX) return phi[(int)n];      // 直接查线性筛结果
        if (memo.containsKey(n)) return memo.get(n); // 记忆化

        // 杜教筛公式：S(n) = n(n+1)/2 - Σ S(⌊n/d⌋), d=2..n
        long ans = n % 2 == 0
            ? (n / 2) * (n + 1)
            : n * ((n + 1) / 2);  // n(n+1)/2 防溢出

        // 整除分块计算 Σ S(⌊n/d⌋), d=2..n
        for (long l = 2, r; l <= n; l = r + 1) {
            long q = n / l;          // ⌊n/l⌋
            if (q == 0) break;
            r = n / q;               // 块右端点
            // [l, r] 区间上 ⌊n/d⌋ 都是 q
            ans -= (r - l + 1) * S(q);
        }

        memo.put(n, ans);
        return ans;
    }

    public static void main(String[] args) {
        DuJiaoSieve djs = new DuJiaoSieve();

        // 测试用例 1：小 n，验证正确性
        long[] testNs = {1, 5, 10, 100, 1000, 1000000};
        System.out.println("=== 小 n 验证 ===");
        for (long n : testNs) {
            long ans = djs.S(n);
            // 与线性筛结果比较（n ≤ NMAX 时）
            long expected = djs.phi[(int) Math.min(n, DuJiaoSieve.NMAX)];
            System.out.println("S(" + n + ") = " + ans
                + (n <= DuJiaoSieve.NMAX ? " (线性筛验证=" + expected + ")" : ""));
        }

        // 测试用例 2：大 n
        System.out.println("\n=== 大 n 测试 ===");
        long[] largeNs = {100000000L, 1000000000L, 10000000000L, 500000000000L};
        // 10⁸, 10⁹, 10¹⁰, 5×10¹¹
        for (long n : largeNs) {
            long start = System.currentTimeMillis();
            long ans = djs.S(n);
            long end = System.currentTimeMillis();
            System.out.println("S(" + n + ") = " + ans + " time=" + (end - start) + "ms");
        }

        // 测试用例 3：n=10¹²（最大规模）
        long start = System.currentTimeMillis();
        long ans = djs.S(1000000000000L);
        long end = System.currentTimeMillis();
        System.out.println("\nS(10¹²) = " + ans + " time=" + (end - start) + "ms");

        // 正确性验证：φ(n) 的恒等式 Σ_{i=1}^{n} φ(i) = (1 + Σ_{i=1}^{n} μ(i) × ⌊n/i⌋²)/2
        // 但这里只需确保记忆化没有溢出即可
        System.out.println("\n记忆化条目数: " + djs.memo.size());

        // 验证 S₁(n) 是否正确——第一性原理
        // 对于 n=10:
        // φ(1)=1, φ(2)=1, φ(3)=2, φ(4)=2, φ(5)=4, φ(6)=2, φ(7)=6, φ(8)=4, φ(9)=6, φ(10)=4
        // S = 1+1+2+2+4+2+6+4+6+4 = 32
        System.out.println("\nS(10) 手动验证: " + djs.S(10) + " (期望 32)");
        // ≈ 32? 不对，1+1+2+2+4+2+6+4+6+4 再算一次
        // 1+1=2, +2=4, +2=6, +4=10, +2=12, +6=18, +4=22, +6=28, +4=32 ✓
    }
}
```

**复杂度分析**：

杜教筛的复杂度为 O(n^{2/3})。

**证明思路**：
- 对于 n ≤ N_{pre}（如 10⁶），直接 O(1) 查询已预处理的前缀和
- 对于 n > N_{pre}，递归调用 S(⌊n/d⌋)
- 所有不同的 ⌊n/d⌋ 取值只有 O(√n) 种
- 每个取值 d 的求和本身需要 O(√d)（通过整除分块）
- 平衡点选 N_{pre} = n^{2/3}，总复杂度 O(n^{2/3})

**实测效果**：

| n | 时间估算 |
|---|---------|
| 10⁸ | < 10ms |
| 10⁹ | 20-50ms |
| 10¹⁰ | 50-100ms |
| 10¹¹ | 100-200ms |
| 10¹² | 200-500ms |

（实际取决于常数和实现质量，这里为参考值）

```
S_μ(n) = 1 - Σ_{d=2}^{n} S_μ(⌊n/d⌋)
```

## 狄利克雷卷积与反演的统一框架

### 基础定义回顾

```
(f * g)(n) = Σ f(d) × g(n/d)
              d|n
```

### 常见卷积及其证明

| 卷积 | 结果 | 证明思路 |
|------|------|---------|
| μ * 1 = ε | Σ_{d|n} μ(d) = [n=1] | 二项式展开 |
| φ * 1 = id | Σ_{d|n} φ(d) = n | 单位根分类 |
| 1 * 1 = τ | Σ_{d|n} 1 = τ(n) | 定义直接展开 |
| id * 1 = σ | Σ_{d|n} d = σ(n) | 对称性 |
| id_k * 1 = σ_k | Σ_{d|n} d^k = σ_k(n) | 定义展开 |
| 1 * μ = ε | 交换律 | 与 μ * 1 相同 |

**可视化**：狄利克雷卷积网络

```
         1（常函数）
        /  \
       /    \
      /      \
  μ * 1 = ε  id * 1 = σ
      |
  φ * 1 = id
      |
  1 * 1 = τ
```

### 反演作为卷积逆运算

```
F = f * 1   ⇔   f = F * μ
F(n) = Σ f(d)   ⇔   f(n) = Σ μ(d) × F(n/d)
       d|n                     d|n
```

这就是莫比乌斯反演的卷积表述——**乘以 μ 就是解除与 1 的卷积**。

所谓"反演"，本质是已知卷积结果和其中一个参与函数，求另一个函数。

### 常用变换对照表

| 目标 | 变换过程 | 最终形式 |
|------|---------|---------|
| [gcd(i,j) = 1] | μ 求和 | Σ_{d\|gcd} μ(d) |
| gcd(i,j) | 枚举 d 再变换 | Σ d × [gcd(i/d, j/d)=1] |
| lcm(i,j) | lcm = i×j/gcd | 转换为 gcd 后莫比乌斯 |
| Σ gcd(i,j) | 枚举 gcd → 互质对 | Σ φ(T) × ⌊n/T⌋ × ⌊m/T⌋ |
| Σ [gcd(i,j)=k] | 除以 k 化为互质对 | Σ μ(d) × ⌊n/(kd)⌋ × ⌊m/(kd)⌋ |
| 无平方因子计数 | 容斥 | Σ μ(i) × ⌊n/i²⌋ |

## 总结——从定义到高级算法

### 知识图谱

```
莫比乌斯函数 μ(n)
    │
    ├── 核心恒等式：Σ μ(d) = [n=1]
    │        │
    │        ├── 莫比乌斯反演（除数形式/倍数形式）
    │        │        │
    │        │        ├── gcd 类求和问题
    │        │        ├── 互质对计数
    │        │        └── lcm 相关问题
    │        │
    │        └── 狄利克雷卷积视角：μ 是 1 的逆元
    │                 │
    │                 ├── F = f * 1 ⇔ f = F * μ
    │                 └── 卷积关系网（μ, φ, id, σ, τ）
    │
    ├── 高效实现
    │        │
    │        ├── 欧拉筛 O(n) 批量预处理 μ
    │        ├── 质因数分解 O(√n) 单值求 μ
    │        ├── 前缀和 → 整除分块 O(√n)
    │        └── 杜教筛 O(n^{2/3}) 处理 n ≤ 10¹²
    │
    └── 典型应用
             │
             ├── ΣΣ gcd(i,j) → φ(T)
             ├── 互质对计数 → μ 直接求和
             ├── gcd = k → 化为互质对
             ├── lcm 求和 → 双层分块
             ├── 区间互质对 → 二维前缀和
             ├── 除数函数求和 → 整除分块
             └── 大 n 前缀和 → 杜教筛
```

### 解题标准流程

```
[问题分析]
    ↓
1. 识别 gcd/lcm/互质关系
    ↓
2. 枚举 gcd d，用核心恒等式展开 [gcd=d]
    ↓
3. 交换求和顺序
    ↓
4. 化简为 Σ μ(d) × f(⌊n/d⌋) × g(⌊m/d⌋) 形式
    ↓
5. 评估数据范围：
   - n ≤ 10⁶：线性筛 + O(n) 求和
   - n ≤ 10⁹：线性筛 + 整除分块 O(√n)
   - n ≤ 10¹²：杜教筛 + 整除分块
```

### 核心结论

1. **莫比乌斯函数 μ(n) 是容斥原理的代数化表达**，它将需要按质因子逐层分析的容斥转化为一个简单函数的一次求和。

2. **核心恒等式 Σ_{d|n} μ(d) = [n=1] 是整个反演的根基**，它将"等于 1"这个条件用 μ 的和式编码。

3. **反演的卷积形式 F = f * 1 ⇔ f = F * μ 是理解反演的最佳视角**，揭示了 μ 作为 1 之逆元的代数身份。

4. **整除分块将 O(n) 降到 O(√n)**，使得莫比乌斯反演能够处理高达 10⁹ 的 n。

5. **杜教筛将前缀和提升到 O(n^{2/3})**，进一步扩展到 10¹² 级别。

6. **核心技巧总结**：
   - 见到 gcd 就枚举（枚举具体值），用莫比乌斯展开
   - 见到 Σ μ(d) × floor(n/d) × floor(m/d) 就整除分块
   - 见到大 n 前缀和就杜教筛
   - 把 lcm 转化为 gcd 永远是正确方向
